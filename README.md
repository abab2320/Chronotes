# 🕒 Chronotes

> **Chronotes = Chrono（时间） + Notes（笔记）**  
> 一个融合任务管理与知识笔记的智能网页应用，助你高效规划时间、沉淀知识，并让 AI 成为你的私人助理。

---

## ✨ 项目简介

**Chronotes** 是一款集「任务管理」与「笔记知识库」于一体的智能网页应用。  
它不仅帮助你管理每日待办、追踪目标，还能让你记录、整理并分析笔记内容。  
内置的 AI 助手会自动为你的笔记生成摘要、提取关键词、甚至根据你的学习记录推荐任务计划。

Chronotes 适合：
- 🧑‍🎓 学生：管理学习任务、整理课程笔记
- 👨‍💻 开发者：记录项目想法、追踪开发计划
- 📚 终身学习者：构建个人知识体系

---

## 🧱 功能概览

### 📅 任务管理（Task Management）
- 创建、编辑、删除任务
- 设置优先级与截止时间
- 按状态、标签或日期筛选任务
- 可视化任务统计（完成率、延误任务分析）
- 日历视图与提醒功能

### 📝 笔记知识库（Notes System）
- Markdown 编辑与实时预览
- 标签分类、笔记搜索与排序
- 上传图片、文档、资源
- 自动生成笔记摘要与关键词
- AI 辅助笔记分析（语义搜索 / 内容问答）

### 🤖 智能助理（AI Integration）
- 自动为笔记生成标签与摘要  
- 根据笔记内容生成任务清单（如学习计划）
- 语义检索（自然语言搜索笔记内容）
- AI 问答：「我在笔记中学过机器学习的梯度下降法吗？」

### 📊 数据可视化
- 周任务完成率统计
- 笔记数量增长趋势
- 知识主题词云

## 🧩 技术栈

| 模块       | 技术选型                                         |
| ---------- | ------------------------------------------------ |
| 前端       | Vue 3 + TypeScript + Vite + Pinia + Element Plus |
| 后端       | Spring Boot 3 / Nest.js（任选其一）              |
| 数据库     | MySQL / PostgreSQL + Redis（缓存）               |
| AI 模块    | OpenAI API / Ollama + LangChain                  |
| 搜索引擎   | Meilisearch / Elasticsearch（可选）              |
| 向量数据库 | FAISS / Milvus（用于语义检索）                   |
| 文件存储   | 本地 / OSS（对象存储服务）                       |
| 部署       | Nginx + Docker（支持生产部署）                   |
